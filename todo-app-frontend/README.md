# Todo App Frontend

Aplicación frontend desarrollada con JavaScript, HTML y CSS, utilizando Vite como herramienta de desarrollo.

---

## Requisitos

- Node.js v18 o superior
- npm v9 o superior

---

## Instalación

1. Entra al proyecto:
cd todo-app-frontend

3. Instala las dependencias:
npm install

---

## Ejecutar el Proyecto

Modo desarrollo:
npm run dev

El servidor se ejecutará en:
http://localhost:5173
